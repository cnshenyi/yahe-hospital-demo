# i五院 互联网医院 VI 原型 v2 — 设计文档

## 项目概述
用 Vue 3 + Vite 构建中山大学附属第五医院互联网医院的高保真原型，包含患者端微信小程序和医生端App两个界面，各4个页面，2套配色方案。

## 技术栈
- Vue 3 (Composition API + `<script setup>`)
- Vite
- Vue Router
- CSS Variables（主题切换）
- 无需UI框架，全部手写CSS以精确还原设计

## 设计要求

### 整体风格
- **高端、真实、专业** — 看起来像真正的医院互联网产品，不是原型/线框图
- **精致细节** — 阴影、圆角、渐变、微动画都要到位
- **两套配色** — 方案A「中山蓝」/ 方案B「科技蓝绿」，通过CSS变量实时切换

### 手机框架
- 尺寸：375 x 812（iPhone X）
- **保留手机状态栏**（时间、信号、电池等）
- 患者端：顶部显示微信小程序的胶囊按钮（··· 和 ⊙）
- 医生端：顶部显示Android状态栏，底部显示Android导航键（◁ ○ □）

### 图标风格
参考六院截图，图标应该是**彩色填充风格**（不是纯线性），有一定的立体感和辨识度。
- 患者端：彩色图标，带浅色圆角背景
- 医生端：绿色/主题色图标，带浅灰圆形背景

## 页面结构

#### P1: 首页
- **状态栏**：iPhone状态栏（时间、信号、WiFi、电池）
- **顶部区域**：
  - 渐变背景（主色→浅色），带淡色装饰纹理（叶子/几何图案）
  - 左上：「i五院」标题（白色，粗体）
  - 右上：微信胶囊按钮（··· | ⊙）

- **公告栏**：
  - 喇叭图标 + 滚动文字「不排队！自助出院办理指南」+ 查看公告链接
