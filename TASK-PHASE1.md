# Claude Code Task: i五院 互联网医院 Vue 原型 — Phase 1

## 目标
用 Vue 3 + Vite 创建中山五院互联网医院的高保真原型。这是给客户演示的VI设计原型，必须看起来像真正的产品，不是线框图。

## 重要：先读设计文档
请先完整阅读 `DESIGN.md`，里面有详细的页面结构、配色方案、组件规划。

## Phase 1 任务

### 1. 项目初始化
```bash
npm create vite@latest . -- --template vue
npm install
npm install vue-router@4
```

### 2. 基础框架
创建以下核心组件：

#### App.vue
- 顶部控制面板（固定在页面顶部，不在手机框内）
  - 主题切换：方案A 中山蓝 / 方案B 科技蓝绿
  - 端切换：患者端 / 医生端
  - 页面切换：对应4个页面按钮
- 手机框架（375x812，居中显示，带圆角和阴影）
- 路由视图

#### PhoneFrame.vue
- iPhone X 外壳样式（圆角、阴影、边框）
- 内部375x812可滚动区域

#### StatusBar.vue
- iOS风格状态栏（用于患者端）：时间 12:45、信号、WiFi、电池80%
- 高度约44px，白色文字在渐变背景上

#### WechatCapsule.vue
- 微信小程序右上角胶囊按钮：「··· | ⊙」
- 半透明白色背景，圆角

#### TabBar.vue
- 底部导航栏，4个tab
- 患者端：首页/健康/消息/我的
- 医生端：工作台/患者/消息/我的
- 选中态：主题色图标+文字，未选中：灰色

### 4. 患者端首页 (Patient Home)
这是最重要的页面，定义整个设计基调。参考六院截图的布局，但用我们自己的配色。

**顶部区域**（渐变背景）：
- 背景：主色渐变 + 淡色装饰纹理（用CSS伪元素画几何图案/叶子轮廓）
- 「i五院」标题（白色粗体，左上）
- 微信胶囊按钮（右上）
- 「沈先生」+ 切换按钮 + 关怀模式按钮

**Banner轮播**：
- 圆角卡片，渐变背景
- 至少2-3张轮播内容（门诊自助缴费、健康体检预约、在线问诊指南）
- 底部圆点指示器
- 自动轮播（3秒间隔）

**公告栏**：
- 浅色背景条
- 喇叭图标 + 文字 + 「查看公告」链接

**核心功能区**：
- 两个大按钮并排：预约挂号 / 在线门诊
  - 带图标、标题、副标题
  - 白色卡片，轻微阴影
- 四个中按钮一行：门诊报到 / 门诊缴费 / 报告查询 / 检查预约
  - 彩色图标 + 文字

**分类Tab区域**：
- 门诊(默认选中) / 住院 / 综合
- 选中态：主题色文字 + 下划线
- 门诊下4个图标：挂号记录 / 医保定点 / 检查检验开单 / 互联网护理

**底部TabBar**

### 5. 图标
所有图标用 SVG inline 实现。风格要求：
- **彩色填充风格**（不是纯线性描边）
- 有一定立体感
- 带浅色圆角方形背景
- 参考医疗类App的图标设计

### 6. 装饰细节
- 顶部渐变区域要有淡色装饰纹理（叶子轮廓、几何线条等），增加层次感
- 卡片要有微妙的阴影（box-shadow: 0 2px 12px rgba(0,0,0,0.08)）
- 圆角统一：大卡片12px，小按钮8px，图标背景12px
- 适当的间距和留白
